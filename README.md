# 哄我上班 😽

一个让你看着时间变成钱的实时薪资时钟。设置好月薪和上下班时间，然后盯着屏幕上的数字不断跳动——每一秒，你都在变得更富有一点。

> 「不是我选择上班，是钱选择了我。」

## 预览

**在线体验：** [哄我上班 😽](https://哄我上班.zxclaw.top)

或者本地直接打开 `index.html`，无需任何构建步骤。

## 功能

- **实时薪资计算** — 精确到小数点后四位，看着每一秒的钱到账
- **双计算模式**
  - 🏢 **上班才赚钱** — 只在工作时段累计薪资，非工作时间显示倒计时+激励语
  - 🌙 **随时都赚钱** — 月薪平摊到整月的每一秒，睡觉也在赚
- **多主题切换** — 极光舞台、霓虹终端、樱桃汽水、深海声呐、黑白杂志、松林晨雾、黑金账本、像素街机，八种带独立纹理、字体、边框和数字效果的个性主题
- **日历排班** — 内置 2024–2026 年中国法定节假日与调休，支持标记公司/个人休假、半天假
- **智能倒计时** — 非工作时段自动显示距离下次上班的倒计时，并附赠毒鸡汤/鸡汤
- **本地存储** — 所有设置自动保存到浏览器，下次打开无需重新配置

## 使用

1. 双击打开 `index.html`
2. 输入月薪、选择上下班时间和午休时长
3. 在日历中标记你的休假/调休安排
4. 点击「进入时钟」开始看着钱增长

## 技术

纯原生 HTML + CSS + JavaScript，单文件零依赖。
